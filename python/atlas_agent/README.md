Atlas Agent (Python)

This package provides a CLI to operate the Atlas application over gRPC using an LLM‑powered multi‑agent workflow.

Usage after installing (editable or wheel):

- Console: `atlas-agent --help`
- Module: `python -m atlas_agent --help`

During development in this monorepo, either install with `pip install -e python/atlas_agent` or set `PYTHONPATH` to include `python/atlas_agent/src` when running from source.
