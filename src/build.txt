OSX build:
0. install xcode, homebrew, qt5, intel c composer (for mkl, ipp, tbb)
1. brew install unrar p7zip git cmake
2. current code repository folder should be named "atlas" and make sure "atlas_others" folder and "atlas" folder are in same directory
3. (setup SSH credential for github), run "build" from 3rdparty folder
4. build atlas/src/CMakeLists.txt

Win build:
0. patch libtiff win32 TIFFFdOpen
1. run "build" from 3rdparty folder
2. build atlas/src/CMakeLists.txt
